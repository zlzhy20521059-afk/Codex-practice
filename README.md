# 计权标准化声压级差（DnT,w）计算程序

## 1. 功能
- 固定 1/3 倍频程频带：100 Hz 到 3150 Hz（16 个频带）。
- 输入列：`声源室声压级`、`受声室声压级`、`Rt`。
- 自动计算每个频带的 `DnT = L1 - L2 + Rt`（保留 1 位小数）。
- 自动计算单值评价量 `DnT,w`（ISO 717-1 参考曲线平移法）。
- 自动计算频谱修正量 `C`、`Ctr`（ISO 717-1 表 4 频谱）。
- 支持从 Excel 多行复制后直接粘贴到表格（`Ctrl+V`）。
- 计算后可一键复制完整表格（TSV），直接粘贴回 Excel。

## 4. Excel 粘贴说明
- 推荐在表格中点击到 `声源室声压级` 列后粘贴。
- 支持两种粘贴格式：
  - 3 列：`声源室声压级 | 受声室声压级 | Rt`
  - 4 列：`频率 | 声源室声压级 | 受声室声压级 | Rt`（频率列会自动识别并跳过）

## 5. 计算依据（ISO 717-1:2013）
- `DnT,w`：按参考曲线在 1 dB 步进平移，满足不利偏差和不超过 32 dB（16 个 1/3 倍频程）。
- `C`、`Ctr`：按 `Cj = XAj - Xw`，其中
  - `Xw = DnT,w`
  - `XAj = -10 lg( sum(10^((Lij - Xi)/10)) )`
  - `Xi = DnT,i`（1 位小数）
  - `Lij` 采用 ISO 717-1 表 4 频谱 1/2。
